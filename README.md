# Alejandra Vergel · Alta Joyería

Sitio web de e-commerce para Alejandra Vergel, especializada en alta joyería de lujo.

## Stack

- **HTML/CSS/JavaScript** puro (sin frameworks)
- **GitHub Pages** para hosting
- **Dominio personalizado:** alejandravergel.com

## Estructura

```
sitio-alejandra-vergel/
├── index.html          # Página principal (hero + catálogo + contacto)
├── products.js         # Catálogo de anillos
├── ANILLOS/            # Carpeta de imágenes y datos
├── README.md
└── .gitignore
```

## Cómo ejecutar localmente

```bash
# Abrir en servidor local (http-server)
npx http-server . -p 8775

# O abrir index.html directamente en el navegador
```

## Deploy a GitHub Pages

El sitio se despliega automáticamente en cada push a `main`.
